# Audit & Logging

Agentee **logs every agent action triggered from chat**, creating a traceable record of what was done, by whom, and when.

## Why it matters

* **Traceability** — reconstruct exactly what the agent did on a user's behalf.
* **Approvals** — the Sales Manager reviews the audit trail for proposals (see [Proposal Builder](../features-and-guides/core-workflows/proposal-builder.md)).
* **Accountability** — pairs with the "AI prepares, humans decide" model so every committed action ties back to an approver.
