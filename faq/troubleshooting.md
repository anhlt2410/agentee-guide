# Troubleshooting

| Symptom | Likely cause | Fix |
|---------|--------------|-----|
| Agentee misread my request | Ambiguous phrasing or intent | Rephrase with specifics; or open the ERP UI to edit directly |
| Agent references stale info | New session expected but old context reused | Start a new session — each new session begins clean |
| Can't log in | ERP SSO issue | Verify ERP portal credentials; access is tied to ERP identity |
| Duplicate lead created | Duplicate check bypassed | Review match rules in [Outreach Campaign Leads](../features/core-workflows/outreach-campaign-leads.md#duplicate-check) |
