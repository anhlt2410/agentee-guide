# Core Workflows

Agentee ships with a set of core sales workflows. Each is triggered by natural
language and follows the "AI prepares, human approves" model.

| Workflow | Trigger | Primary role |
|----------|---------|--------------|
| [Proposal Builder](proposal-builder.md) | "Build a proposal for…" | Sales Rep |
| [Lead Creation](lead-creation.md) | Scan an image / photo | Sales Rep |
| [Outreach Campaign Leads](outreach-campaign-leads.md) | "Enrich and reach out to…" | Sales Agent |

Each workflow page follows the same shape: **goal → roles → stages/steps →
example → gotchas.**
