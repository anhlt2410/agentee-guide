# Prompting Tips

Agentee understands plain language, but clear requests get cleaner results.

## Include the specifics

Put the key details in the message — quantity, item, target record, and timing.

> Create an order for **50 iced milk coffees** for the **Q1 store**, delivery
> **tomorrow at 8:00 AM**.

## Ask for comparisons directly

> How is **this month's** revenue compared to **last month**?

## One intent per message

Keep a single action per message when you can; chain follow-ups across turns
(Agentee remembers context within the session).

{% hint style="info" %}
When an action changes data, Agentee shows you a draft to approve. If a draft
looks wrong, correct it in a follow-up message rather than starting over.
{% endhint %}
